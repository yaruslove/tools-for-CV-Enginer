Напиши скрипт он должен прочитать все файлы в директории где находится разметка детекции yolo txt labels и выбрать только те названия которые внутри не пуыте тоесть только те файлы где внутри есть какйо-то текст. Файл .txt в labels может быть но он может быть пустой с таким префиком копировать изображения не нужно.
src_labels:
Дальше он согласно этому списку убирая окончания расширения .txt должен выбрать из другой папки изображения у которых одинаковый prefix 


Пример он октрывает файл pexels-photo-15888226.txt если он не нулевого размера и там есть текст то он сохраняет себе внутри в список добалвяет pexels-photo-15888226.

Потом он по этому списоку prefix пробегается внутри src_images:
и копирует только те картики с которыми совпадает prefix например он должен скопировать pexels-photo-15888226.jpeg в папку dst_images

Скрипт читает config.yaml внутри которого
```
src_labels: "- путь где директория labels и хранятся txt files часть файлов пыстые часть нет"

src_images: "путь откуда нужно копировать картинки"
dst_images: "путь куда копировать картинки"
```

Если резюмировтаь скрипт должен скопировать фото из src_images => в dst_images согласно не пустым текстовым файлам src_labels  по префиксу
